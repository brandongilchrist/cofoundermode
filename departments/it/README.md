# IT Department

## Charter
The IT department manages internal infrastructure, system administration, monitoring, and technical support for all CoFounderMode systems. We keep the lights on — DNS, hosting, CI/CD infrastructure, database administration, and agent runtime environments.

## Department Head
**CIO Agent** — Reports to CEO

## KPIs
- **System uptime**: Target 99.5%
- **Incident response**: Target <30 minutes for infrastructure issues
- **Deployment success rate**: Target >98%
- **Infrastructure cost optimization**: Keep cloud spend within 10% of budget
- **Backup compliance**: Daily backups with weekly restore verification

1. Set up production infrastructure (AWS/GCP via Terraform)
2. Configure Paperclip server deployment and monitoring
3. Establish database backup and disaster recovery procedures
4. Set up CI/CD infrastructure (GitHub Actions runners)
5. Configure LobsterBoard and agent runtime environments
