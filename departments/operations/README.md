# Operations Department

## Charter
The operations department keeps the company running smoothly. We manage dashboards, reports, SLAs, cross-department coordination, and operational health monitoring. We are the connective tissue between all departments.

## Department Head
**COO Agent** — Reports to CEO

## KPIs
- **Operational uptime**: Target 99.5% across all systems
- **Report delivery**: 100% on-time delivery of daily/weekly reports
- **Cross-dept coordination SLA**: All inter-department requests resolved within 4 hours
- **Dashboard accuracy**: All LobsterBoard metrics within 5 minutes of real-time
- **Agent health**: All agents responding to >95% of heartbeats

1. Set up LobsterBoard mission control dashboard with key operational metrics
2. Establish daily standup and weekly report workflows
3. Define SLAs for every department interaction
4. Build budget tracking dashboard (per-agent, per-department)
5. Create operational runbooks for common scenarios
