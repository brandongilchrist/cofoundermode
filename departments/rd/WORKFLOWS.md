# R&D Workflows

## 1. Technology Evaluation
```
Trigger: New technology identified by any agent or board request
Steps:
  1. CRO receives evaluation request with context
  2. Researches the technology using Perplexity Computer
  3. Produces evaluation report: capabilities, costs, risks, integration complexity
  4. Compares against current stack and alternatives
  5. Recommends adopt / trial / assess / hold
  6. Routes to CEO for strategic alignment review
```

## 2. Monthly Research Report
```
Trigger: Last Friday of each month
Steps:
  1. CRO aggregates all research findings from the month
  2. Identifies top 3 trends relevant to CoFounderMode
  3. Proposes actionable recommendations
  4. Produces a research report artifact
  5. Presents findings to CEO and relevant department heads
```

## 3. Competitive Intelligence
```
Trigger: Bi-weekly
Steps:
  1. CRO monitors competitor activity (product launches, funding, hiring)
  2. Tracks AI orchestration market developments
  3. Identifies threats and opportunities
  4. Updates competitive landscape document
  5. Alerts CEO if significant competitive move detected
```
