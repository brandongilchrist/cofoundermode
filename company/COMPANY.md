# CoFounderMode

**Parent Organization:** Gilchrist Research
**Founded:** 2026
**Type:** AI-Native Autonomous Company Builder

---

## Identity

CoFounderMode is the autonomous company infrastructure platform built by Gilchrist Research. We build the operating system that lets AI agents run entire companies — from engineering and security to marketing and finance — with human founders maintaining strategic oversight through a board-level control plane.

We are not a chatbot company. We are not an AI wrapper. We build **companies that run themselves** using coordinated fleets of specialized AI agents orchestrated by Paperclip, operating across multiple runtimes (OpenClaw, NanoClaw, Claude Code, Codex, Goose, Perplexity Computer), and governed by explicit policy documents that agents can read, interpret, and follow.

## Mission

To eliminate the operational overhead of running a company by replacing every repeatable business function with autonomous AI agents — so human founders can focus purely on vision, strategy, and the work only humans can do.

## Values

### 1. Autonomy Over Automation
We don't build scripts that run. We build agents that *think*, *decide*, and *execute* — within explicit guardrails. The difference between automation and autonomy is judgment.

### 2. Transparency by Default
Every agent decision is logged. Every tool call is auditable. Every budget dollar is tracked. If an agent can't explain why it did something, it shouldn't have done it.

### 3. Governance is a Feature
Governance isn't bureaucracy — it's the immune system that lets agents operate safely at scale. Board approval thresholds, budget hard stops, and red lines exist to protect the company from itself.

### 4. Tool-Agnostic Architecture
We never marry a tool. Every integration uses the adapter pattern. Linear today, custom PM tomorrow. Slack today, custom comms tomorrow. The orchestration spine (Paperclip) doesn't care what's plugged in.

### 5. Memory is Sacred
Agents without memory are glorified API calls. Every agent maintains persistent memory (MEMORY.md), every decision is logged, and the company's knowledge graph grows with every heartbeat cycle.

### 6. Ship Daily, Break Nothing
TDD-first. Tests before code. Code review before merge. Security scan before deploy. The autonomous pipeline is only as good as the gates that protect it.

## Operating Principles

- **Async-first**: All agent communication happens through structured channels (tickets, PRs, reports) — never ephemeral chat
- **Goal ancestry**: Every task traces back to Company Mission → Department Mission → Project Goal → Task. No orphan work.
- **Heartbeat-driven**: Agents don't wait to be told. They wake on schedule, check their queue, execute, and delegate.
- **Escalation over assumption**: When in doubt, escalate to the next level. A wrong escalation costs minutes; a wrong autonomous decision costs days.
- **Build-then-buy reversal**: Start with third-party tools (Linear, Slack, Pinecone), then systematically replace them with custom-built alternatives as competitive moats form.
