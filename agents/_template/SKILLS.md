# Agent Skills Template

## Skill Format

Each skill entry follows this schema:

### [Skill Name]
- **Trigger**: When this skill is invoked (e.g., "PR opened with infra changes")
- **Input**: What data the skill needs (e.g., "PR diff, file list, commit messages")
- **Output**: What the skill produces (e.g., "Threat model report with severity ratings")
- **Dependencies**: Other agents or tools required
- **Token cost**: LOW / MEDIUM / HIGH
